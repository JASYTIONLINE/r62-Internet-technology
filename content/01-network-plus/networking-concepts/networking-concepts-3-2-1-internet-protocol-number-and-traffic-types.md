## IP Protocol Numbers and Traffic Types

At the end of this episode, I will be able to:    

1. Identify and explain the purpose of IP protocol numbers and traffic types, given a scenario.    

Learner Objective: *Identify and explain the purpose of IP protocol numbers and traffic types, given a scenario.*   

Description: In this episode, the learner will examine IP protocol numbers used in IP headers to identify upper level protocols. We will also explore traffic types such as unicast, broadcast and more.

--------  

* Internet Protocol (IP) types
  + Internet Control Message Protocol \(ICMP\)
    - IP Protocol ID - 1
  + Internet Group Management Protocol \(IGMP\)
    - IP Protocol ID - 2
  + Transmission Control Protocol \(TCP\)
    - IP Protocol ID - 6
  + User Datagram Protocol \(UDP\)
    - IP Protocol ID - 17
  + Generic Routing Encapsulation \(GRE\)
    - - IP Protocol ID - 
  + Internet Protocol Security \(IPSec\) 
    - Authentication Header \(AH\)
	  * IP Protocol ID - 51
	- Encapsulating Security Payload \(ESP\)
	  * IP Protocol ID - 50
* Traffic types
  + Unicast - one-to-one communication
  + Multicast - one-to-many communication
  + Broadcast - one-to-all communication
  + Anycast - one-to-the-best communication
