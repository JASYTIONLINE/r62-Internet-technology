Learner Objective: *Learn about baseline metrics in networking today*    

Description: In this episode, you will learn about the importance of metrics for monitoring and maintaining your networking environment. You will also learn about the importance of baseline metrics and how they can be used in your organization.   

-----------

* Metrics refer to measurable quantities or parameters used to assess, quantify, and evaluate various aspects of IT processes, systems, services, and performance. 
* Metrics play a crucial role in monitoring, managing, and improving IT operations. These measurements provide valuable insights into the efficiency, effectiveness, and overall health of IT activities. 
* Here are several types of IT metrics:
 - Performance metrics
 - Availability metrics
 - Reliability metrics
 - Security metrics 
 - Capacity metrics 
 - SLA metrics
 - Cost metrics 
* Baseline metrics - baseline metrics refer to a set of established, stable reference points that represent the normal or expected performance, behavior, or state of a system or process under typical conditions. 
* These metrics serve as a baseline against which deviations or anomalies can be identified and analyzed. Establishing baseline metrics is crucial for monitoring, managing, and troubleshooting IT systems effectively. 
* Once baseline metrics are established, it is recommended to put an anomaly alerting/notification system in place. This fosters the use of automation in network troubleshooting and a more proactive approach to maintaining the network.
