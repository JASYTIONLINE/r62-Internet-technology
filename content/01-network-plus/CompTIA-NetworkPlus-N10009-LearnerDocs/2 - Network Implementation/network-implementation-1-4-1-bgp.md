1. Border Gateway Protocol (BGP)

Learner Objective: *Learn about the Border Gateway Protocol (BGP)*    

Description: In this episode, you will learn about the exterior gateway protocol called Border Gateway Protocol (BGP). You will learn key theory about this important protocol, and you will even learn about basic configurations that make this protocol function.    

-----------

* Exterior Gateway Protocol for sharing route prefixes between Autonomous Systems 
* Considered a path vector protocol 
* Makes best path decisions based on a wide variety of metrics called attributes - this permits service providers to easily manipulate the flow of Internet traffic 
* BGP uses iBGP to carry prefix information within Autonomous Systems and eBGP to carry prefix information between these Autonomous Systems 
* You configure BGP with the manual establishment of neighbors, and the careful injection of network prefixes 
* BGP is easily extended to support new protocols  - such as IP version 6
