1. Virtual Local Area Networks (VLANs)

Learner Objective: *Learn about VLANs and their usage in modern networks*    

Description: In this episode, you will learn about virtual local area networks and how they are used and configured in modern networks.   

-----------

* A Virtual Local Area Network (VLAN) is a network infrastructure that allows the grouping of devices on a local area network (LAN) based on logical criteria rather than the physical layout of the network.
* VLANs provide logical segmentation within a physical network, allowing an organization to divide its network into smaller, more manageable segments.
* In a traditional LAN, broadcasts are sent to all devices in the broadcast domain, leading to increased network traffic. VLANs limit the broadcast domain to the devices within the same VLAN, reducing unnecessary traffic and improving network efficiency.
* VLANs enhance network security by isolating different groups of users or departments. Traffic between VLANs typically requires routing, and network administrators can implement access control policies to regulate communication between VLANs. This helps contain and control the spread of malicious activity within the network.
* VLANs provide flexibility in network design and facilitate easier network changes.
