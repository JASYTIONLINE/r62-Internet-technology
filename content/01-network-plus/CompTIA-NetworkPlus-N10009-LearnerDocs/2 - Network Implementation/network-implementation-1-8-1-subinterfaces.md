## Subinterfaces

1. Subinterfaces

Learner Objective: *Learn about subinterfaces on network devices*    

Description: In this episode, you will learn about the function and configuration of subinterfaces on network devices.    

-----------

* Subinterfaces allow you to carve multiple logical interfaces out of a single physical interface. 
* Different vendors will have different configuration approaches and limits on the number of subinterfaces that can be created. 
* Cisco allows a massive number of subinterfaces to be created on a single physical interface. 
* The configuration is simple - using a sub interface identifier following the physical router interface - for example "interface gigabitethernet 0/0.100"
