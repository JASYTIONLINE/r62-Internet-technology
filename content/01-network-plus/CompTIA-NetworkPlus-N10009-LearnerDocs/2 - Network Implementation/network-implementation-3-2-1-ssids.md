## SSIDs

1. Service Set Identifier (SSID)

Learner Objective: *Learn about the concepts of BSSID, SSID, and ESSID in modern wireless networks*    

Description: In this episode, you will learn about the concept of basic service set identifiers, service set identifiers, and extended service set identifiers in wireless networks.    

-----------

* Wireless networks require a means of identification for the network - there are several different approaches to this in use today. 
* The basic service set identifier (BSSID) is used for the most basic infrastructure mode network - this is a network that consists of one wireless access point and its wireless clients. In this simple setup, the network is identified by the MAC address of the wireless access point. 
* To simplify naming when compared to a MAC address, the standards makers created the service set identifier. This is often simply considered a network name - it is a 32-bit identification string that is inserted into the header of each frame processed by the wireless access point. With this configuration, every WiFi device must share this same SSID in order to communicate in the single network. By default the wireless access point broadcasts the SSID. 
* What about a network that features many wireless access points? This is the reason for the extended service set identifier (ESSID). This is the SSID applied to the many access points. Clients can change the wireless access point they are associated with through a process called roaming.
