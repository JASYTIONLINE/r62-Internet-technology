1. Enhanced Interior Gateway Routing Protocol (EIGRP)

Learner Objective: *Learn about the Enhanced Interior Gateway Protocol (EIGRP)*    

Description: In this episode, you will learn about the Enhanced Interior Gateway Protocol (EIGRP) developed by Cisco Systems. You will learn the basics theory of this protocol, and you will also learn about the basic configurations that are required for its operation.    

-----------

* EIGRP is an enhanced version of the IGRP protocol invented by Cisco Systems 
* For decades, EIGRP was considered a proprietary, non-standard routing protocol; since then, Cisco has made the protocol an open standard 
* EIGRP is considered a hybrid routing protocol - using some of the characteristics of earlier distance vector protocols and combining those with the enhancements found in link state routing protocols 
* EIGRP uses a Diffusing Update Algorithm (DUAL) in order to quickly calculate a loop free topology for routing 
* EIGRP offers a feasible successor concept that permits for very fast convergence upon the loss of a primary best path in the routing infrastructure 
* EIGRP functions through the use of five packet types - Hello/Acks, Updates, Queries, Replies, and Requests
