Description: In this episode, you will learn about the different network types that are found with wireless technologies.    

-----------

* There are several different network types that are used with wireless. We need to be aware of the following types for Network+ requirements. 
* Ad hoc - a decentralized and temporary communication infrastructure formed spontaneously among wireless devices, enabling them to connect directly with one another without the need for a pre-existing centralized network infrastructure or router. These networks are often created on-the-fly for specific purposes, such as ad hoc meetings or collaborative tasks, and dissolve once the need for connectivity diminishes.
* Point-to-point - a communication setup where two devices or nodes establish a direct and dedicated link between them for the purpose of data transmission. In this configuration, data travels directly from one endpoint to the other, creating a dedicated and efficient connection without the need for a broader network infrastructure.
* Infrastructure -  a communication system where devices connect to a central infrastructure, such as a wireless router or access point, to facilitate communication and data exchange. This centralized network architecture enables multiple devices to interact within a defined coverage area, providing a stable and organized framework for wireless connectivity.
* Mesh - a decentralized communication structure where interconnected devices collaborate to relay data, creating a dynamic and self-configuring network. In this topology, each node in the network can communicate with neighboring nodes, enhancing reliability, coverage, and adaptability, often resulting in increased resilience and efficiency compared to traditional point-to-point or infrastructure-based networks.
