Description: In this episode, the learner will examine the process of authorization and access control models such as Discretionary Access Control, Role-based Access Control and more.

--------  

* What is authorization?
	+ Granting or denying specific permissions and access rights to authenticated users or entities, controlling their actions within a system or resource.
* How are these privileges and permissions granted?
	+ Access control systems that implement common access control models
* What types of access control models are we talking about?
	+ Discretionary Access Control - a security model where data owners determine and manage access permissions to their resources.
	+ Mandatory Access Control - a security model where access permissions are determined by a central authority based on predefined security labels or classifications
	+ Role-based Access Control - s a security model where access permissions are determined by a central authority based on predefined security labels or classifications
	+ Rule-based access control - a security method where access decisions are determined by a set of explicitly defined rules or conditions
