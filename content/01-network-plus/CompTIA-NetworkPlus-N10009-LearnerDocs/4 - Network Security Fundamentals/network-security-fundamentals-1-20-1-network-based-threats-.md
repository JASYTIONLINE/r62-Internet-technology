Network-Based Threats
======================

+ Wired
  - Physical access
    + Unauthorized Access
    + Tampering
    + Tapping
  - VLAN Hopping
  - ARP Poisoning/Spoofing
  - Rogue Devices
    + Switches
    + Access Points
  - No Segmentation
  - No Encryption
  - No Authentication
+ Wireless
  - Default Creds
  - Weak/No Encryption
    + WEP
    + WPA
    + WPA2 (KRACK)
  - Disassociation
  - Jamming
  - Rogue AP
  - Evil Twin
+ Bluetooth
  - Blue Jacking
    + Unsolicited messages
      - Spam
      - Malware
  - Bluesnarfing
    + Unauthorized pairing of bluetooth devices
      - Allows attacker to steal data
  - Bluebugging
    + Full takeover of target device
  - Bluebourne
    + Full takeover of target device
  - Car Whispering
    + Attack used to gain access to bluetooth enabled car stereo systems
      - Eavesdrop on phone conversations
      - Inject audio
