Learner Objective: *Identify and explain deception and disruption technologies, given a scenario*    

Description: In this episode, the learner will examine deceptive technologies used to lure potential attackers, such as honeypots and honeynets.

--------  

* Deception and disruption technologies
	+ Honeyfile
		- A file or document placed on a network as bait to attract
		- Enables security professionals to attract and detect unauthorized access or exfiltration attempts.
		- This information can be studied to identify the techniques and potential intentions of bad actors without risking actual resources.
	+ Honeytoken
		- A piece of fake or deliberately weak data placed within a system or network to act as a decoy.
		- Enables security professionals to spot potential unauthorized activity or security breaches.
		- This system information can be studied to identify techniques and potential intentions without risking actual network assets.
	+ Honeypot
		- A deceptive system or network designed to attract and trap potential attackers
		- Enables security professionals to monitor the activities of bad actors/threat actors
		- This system information can be studied to identify techniques and potential intentions without risking actual network assets.
	+ Honeynet
		- A network of honeypots designed to simulate a larger, more enticing target to lure and study attackers.
		- Enables security professionals to monitor and study the activities of bad actors/threat actors without risking actual network assets.
